# PasswordGeneration

Состав проекта
--------------

Класс PasswordData - является хранилищем данных, необходимых для ответа на запрос пользователя. 
Класс GetPasswordData - класс, содержащий в себе единственный публичный метод (является
статическим), который изменяет свойства объекта класса PasswordData, генеририя необходимые
данные. 

Документация
------------

В проекте подключена документация Swagger, которую можно найти при запуске приложения, добавив в
url адрес "/swagger", после чего автоматически откроется документация по работе с API. 

API
---

Сервис содержит следующие запросы к API:
get - позволяет получить сгенерированные пароль, соль и хеш, на основе входных данных (длина 
пароля и длина хеша). Пример запроса: <адрес сервиса>/PassGeneration?pass=xx&salt=xx
